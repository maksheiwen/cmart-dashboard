# 🏬 Cmart Changlun Analytics Dashboard

An interactive dashboard for Cmart Changlun that combines text analytics (vendor & customer interviews) and social network analysis (TikTok hashtags) to support a data-driven loyalty program. Features include sentiment analysis, topic modeling, SNA visualization, influencer identification, model comparison, dark mode, and downloadable CSV reports. Built with Streamlit, Plotly, Malaya, and scikit‑learn.

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run dashboard/app_streamlit.py
